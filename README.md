# 칵테일 레시피 서비스

## 프로젝트 소개

- 칵테일 레시피를 쉽게 찾아보고 직접 등록할 숙 있는 서비스

## 핵심 기능

- 사용자가 직접 칵테일 레시피를 등록
- 재료와 태그 기준으로 레시피를 분류 및 검색
- 레시피별 재료와 용량에 따라 예상 도수 확인

## 주요 화면

- 메인화면
- 칵테일 레시피 상세화면
- 칵테일 레시피 등록 화면
- 회원가입, 로그인, 마이페이지

## 화면별 필요 데이터

### 메인화면

- 레시피 목록 : 칵테일 이름, 대표 칵테일 사진, 태그, 좋아요, 댓글
- 오늘의 추천 칵테일 : 칵테일 이름, 태그

### 칵테일 레시피 상세화면

- 칵테일 이름, 대표 칵테일 사진, 칵테일 설명, 만드는 방법, 재료

### 칵테일 레시피 등록 화면

- 칵테일 이름, 대표 칵테일 사진, 칵테일 설명, 만드는 방법, 재료, 태그

### 회원가입

- 아이디(이메일), 비밀번호, 이름, 성별, 닉네임, 전화번호, 주소, 생년월일

### 로그인

- 아이디(이메일), 비밀번호

### 마이페이지

- 아이디(이메일), 비밀번호, 이름, 성별, 닉네임, 전화번호, 주소, 생년월일
